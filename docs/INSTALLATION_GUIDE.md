### Setting Up the Web Dashboard
1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Build the React App**:
   ```bash
   npm run build
   ```

3. **Start the Express Server**:
   ```bash
   node src/server.js
   ```

4. **Access the Dashboard**:
   Open your web browser and navigate to `http://localhost:3000`.
