# Shop System

## Configuration
```json
{
    "items": [
        {
            "id": "Base.BaseballBat",
            "name": "Baseball Bat",
            "cost": 100,
            "quantity": 1
        }
    ]
}
```

## Commands
- `!shop` - View available items
- `!buy <item>` - Purchase an item
- `!points` - Check your balance

## Integration
The shop system integrates with:
- Points system
- RCON commands
- Game inventory
